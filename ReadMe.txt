Билетная Система для кинотеатра.
Данное десктопное приложение выполняет функцию покупки-продажи билетов на сеансы кино.
Фреймворк: Pyqt5
БД: sqlite / postgresql (обе версии находятся в папках sqlite_version и postgresql_version соответственно)
Реализованые возможности:
-Создание / удаление кинотеатров
-Создание / изменение / удаление залов и сеансов кино в залах
-Продажа билетов на сеанс с выбором мест
-При продаже билета формируется pdf файл с билетом на выбранное место

За пароль владельца отвечает константа PASSWORD в классе OwnerPassDialog в SignInWidget.py

В уже существующем кинотеатре "Кинотеатр3000" для sqlite версии пароль 12345
Если вы используете postgresql, вы должны указать данные для подключения к бд в файле db/db_credentials.py
А также создать соответствующие таблицы в своей базе данных согласно диаграмме ER Diagram.jpeg
