Билетная Система для кинотеатра.
Данное десктопное приложение выполняет функцию покупки-продажи билетов на сеансы кино.
Фреймворк: Pyqt5
БД: sqlite
Реализованые возможности:
-Создание / удаление кинотеатров
-Создание / изменение / удаление залов и сеансов кино в залах
-Продажа билетов на сеанс с выбором мест
-При продаже билета формируется pdf файл с билетом на выбранное место
В уже существующем кинотеатре "Кинотеатр3000" пароль 12345
